# Odin Recipes

## Project Overview

This project is a simple recipe website built as part of The Odin Project Foundations Course. It demonstrates basic HTML skills by creating a multi-page website that links to individual recipe pages.

The goal of this project is to practice creating HTML files, organizing folders, using links for navigation, and structuring content with headings, lists, and images.

## Features

- A homepage with links to multiple recipes
- Individual recipe pages that include:
  - An image of the dish
  - A description section
  - An ingredients list
  - Step-by-step instructions
- Navigation links between pages

## Project Structure

```
odin-recipes/
│
├── index.html
│
└── recipes/
    ├── recipe1.html
    ├── recipe2.html
    └── recipe3.html
```

## Skills Demonstrated

- Basic HTML boilerplate structure
- Headings and paragraphs
- Ordered and unordered lists
- Relative links between pages
- Image embedding
- Folder organization

## How to Use

1. Clone or download this repository.
2. Open the `index.html` file in a web browser.
3. Click the recipe links to navigate through the site.
